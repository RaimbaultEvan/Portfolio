Création d'un site web pour une société de location de gîtes fictive

Projet réalisé dans le cadre de mon BTS SIO SISR

Présentation du projet

Ce projet consistait à concevoir un site web pour une entreprise de location de gîtes fictive, comprenant la création des pages internet composant ce site et leurs fonctionnement.

L'objectif était de créer un site web fonctionnel, facile de navigation et pouvant se rapprocher au maximum d'un potentiel site commercial.
